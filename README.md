## Project Title
HAZE WATCH basic SPA web app to check out the current haze levels based in Singapore

## Tech/Framework used
**Built with**
- Bootstrap 4
- JavaScript
- HTML5
- jQuery

## Features
- check 1hr PSI based on PM2.5 hourly
- normal PM2.5 reader
- 24hr haze
- able to change region to fit
- Main BG UI changes to fit the haze situation
